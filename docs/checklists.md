# Чеклісти для тестування

## Модуль: Перегляд розкладу групи (публічна частина)

---

### Функціональність

- [ ] Список груп відображається на сторінці при завантаженні
- [ ] Після вибору групи відображається її розклад
- [ ] Розклад відображається з розбивкою по днях тижня
- [ ] Перемикач «Непарний / Парний тиждень» присутній на сторінці
- [ ] Після натискання «Парний тиждень» розклад оновлюється і показує пари парного тижня
- [ ] Після натискання «Непарний тиждень» розклад повертається до непарного тижня
- [ ] Для кожного заняття відображається: назва предмету, ПІБ викладача, час, аудиторія, тип заняття
- [ ] При виборі іншої групи розклад оновлюється без перезавантаження сторінки
- [ ] Якщо на обраному тижні немає пар — відображається відповідне повідомлення або порожній стан

---

### Негативні сценарії

- [ ] При виборі групи без жодних пар сторінка не падає і показує коректний порожній стан
- [ ] При відсутності пар лише на непарному тижні — парний тиждень все одно відображається коректно
- [ ] При відсутності пар лише на парному тижні — непарний тиждень відображається коректно
- [ ] При швидкому багаторазовому переключенні між парним та непарним тижнем сторінка не зависає
- [ ] При швидкій зміні групи кілька разів підряд — відображається розклад останньої обраної групи
- [ ] Якщо у заняття відсутня аудиторія — відображення не ламається (немає undefined або порожнього блоку без підпису)

---

### UI/UX

- [ ] Назви груп у списку читабельні та повні (без обрізання)
- [ ] Активний тип тижня (парний / непарний) візуально виділений
- [ ] Дні тижня без занять відображаються коректно (або приховані, або з позначкою «немає пар»)
- [ ] Інтерфейс коректно відображається на ширині екрана 1280px
- [ ] Інтерфейс коректно відображається на ширині екрана 768px (планшет)
- [ ] Інтерфейс коректно відображається на ширині екрана 375px (смартфон)
- [ ] Відсутній горизонтальний скрол на мобільному пристрої
- [ ] Шрифт і розміри елементів читабельні без збільшення масштабу
